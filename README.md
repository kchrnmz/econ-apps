# Economics Interactive Teaching Tools

Interactive web applications for teaching economics concepts. All tools are freely available and designed for classroom use.

## Available Tools

### Production Functions (3D Visualization)

Interactive 3D visualization of production functions showing relationships between inputs (labor and capital) and output.

**Features:**
- 3D rotatable production surface
- Multiple production functions: Cobb-Douglas, CES, Linear (Perfect Substitutes), Fixed-Proportions (Perfect Complements)
- Isoquant analysis and marginal product visualizations
- Synchronized 2D/3D views with interactive controls

**Concepts:** Production functions, isoquants, marginal products, marginal rate of technical substitution (MRTS), returns to scale

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/production_standalone.html)**

---

## About

These tools use modern web technologies (React, Three.js, Canvas) to provide interactive visualizations of economic concepts. No installation required - all tools run directly in your browser.
