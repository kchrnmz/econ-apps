# Economics Interactive Teaching Tools

Interactive web applications for teaching economics concepts. All tools are freely available and designed for classroom use.

## Available Tools

| # | Tool | Key Concepts |
|---|------|--------------|
| 1 | [Utility Functions](#1-utility-functions-3d-visualization) | Utility surfaces, indifference curves, MRS |

| 3 | [Expenditure Minimization (EMP)](#3-expenditure-minimization-problem-emp) | Hicksian demand, duality |
| 4 | [Welfare Analysis](#4-welfare-analysis-compensatingequivalent-variation) | CV, EV, consumer surplus |

### 1. Utility Functions (3D Visualization)

Interactive 3D utility function visualization with synchronized 2D analysis views.

**Features:**
- 3D rotatable utility surface U(x,y)
- Multiple utility functions: Cobb-Douglas, Perfect Substitutes, Perfect Complements, CES, Quasilinear
- Three analysis modes: IC Map, Fix Y (U vs X), Fix X (U vs Y)
- Slicing planes synchronized with analysis mode

**Concepts:** Utility functions, indifference curves, marginal rate of substitution (MRS), marginal utilities, preference visualization

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/util_standalone.html)**

---

### 2. Utility Maximization Problem (UMP)

Interactive consumer optimization visualization demonstrating utility maximization subject to budget constraints.

**Features:**
- Multiple utility functions: Cobb-Douglas, Perfect Substitutes, Perfect Complements, CES, Quasilinear
- Consumer choice visualization with budget lines and indifference curves
- Demand curve and Engel curve visualizations
- Tangency condition verification

**Concepts:** Utility maximization, budget constraints, indifference curves, tangency conditions, demand curves, Engel curves

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/ump_standalone.html)**

---

### 3. Expenditure Minimization Problem (EMP)

Dual to UMP - visualizes how consumers minimize expenditure to achieve a target utility level.

**Features:**
- Expenditure minimization with target utility constraints
- Hicksian (compensated) demand curves
- Expenditure function visualization
- Tangency condition demonstrations

**Concepts:** Expenditure minimization, Hicksian demand, compensated demand, expenditure functions, duality theory

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/emp_standalone.html)**

---

### 4. Welfare Analysis (Compensating/Equivalent Variation)

Interactive visualization of welfare measures in consumer theory, showing how price changes affect consumer welfare.

**Features:**
- Three welfare measures: Compensating Variation (CV), Equivalent Variation (EV), Consumer Surplus Change
- Consumption space plots with budget lines and indifference curves
- Demand curve visualization with shaded welfare areas
- Welfare mode toggle for focused analysis

**Concepts:** Compensating variation, equivalent variation, consumer surplus, welfare analysis, price changes

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/comp_standalone.html)**

---

### 5. Slutsky Decomposition

Interactive visualization demonstrating how price changes affect consumer choices through income and substitution effects.

**Features:**
- Full Slutsky decomposition into substitution and income effects
- Walrasian (ordinary) and Hicksian (compensated) demand curves
- Visual decomposition in consumption space
- Multiple utility functions: Cobb-Douglas, Perfect Substitutes

**Concepts:** Slutsky decomposition, substitution effect, income effect, Walrasian demand, Hicksian demand

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/slutsky_standalone.html)**

---

### 6. Production Functions (3D Visualization)

Interactive 3D visualization of production functions showing relationships between inputs (labor and capital) and output.

**Features:**
- 3D rotatable production surface
- Multiple production functions: Cobb-Douglas, CES, Linear (Perfect Substitutes), Fixed-Proportions (Perfect Complements)
- Isoquant analysis and marginal product visualizations
- Synchronized 2D/3D views with interactive controls

**Concepts:** Production functions, isoquants, marginal products, marginal rate of technical substitution (MRTS), returns to scale

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/production_standalone.html)**

---

### 7. Cost Minimization Problem (CMP)

Visualizes cost minimization for firms - finding the least-cost combination of inputs to produce a target output level.

**Features:**
- Cost minimization with production constraints
- Four production functions: Cobb-Douglas, CES, Perfect Substitutes, Perfect Complements
- Conditional factor demand curves
- Cost expansion path visualization
- Tangency condition (MRTS = w/r)

**Concepts:** Cost minimization, isocost lines, conditional factor demands, cost functions, expansion paths, input substitution

**[Launch Tool →](https://kchrnmz.github.io/econ-apps/cost_standalone.html)**

---

## About

These tools use modern web technologies (React, Three.js, Canvas) to provide interactive visualizations of economic concepts. No installation required - all tools run directly in your browser.
